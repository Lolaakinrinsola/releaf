# Getting Started with Create React App

This project shows a sample dashboard for a user

## Available Scripts

In the project directory, you can run:

### `npm install`
This installs all dependancies for the site.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### Built with
- [React](https://react.dev/) -JavaScript library for building user interfaces that allows developers to create dynamic and reusable UI components.
- [Tailwind](https://tailwindcss.com/) -utility-first CSS framework that simplifies web development by providing a set of pre-designed utility classes for quickly styling HTML elements. 



### Deployment

This site has been deployed on vercel at https://releaf2.vercel.app/
